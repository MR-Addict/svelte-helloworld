# Svelte Hello World

## 1. Tailwindcss Rules

| Key        | Value    |
| :--------- | :------- |
| background | black    |
| light text | gray-300 |
| dark text  | gray-500 |

## 2. Deploy

Clone repository:

```bash
git clone https://github.com/MR-Addict/svelte-helloworld.git
```

Install packages:

```bash
npm install
```

Build page:

```bash
npm run build
```

Preview page

```bash
npm run preview
```
